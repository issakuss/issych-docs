issych.dataclass.Dictm#

class issych.dataclass.Dictm[source]#
__init__(*args, **kwargs)[source]#

Notes

組み込みの辞書型と同じ機能を持っています。 組み込みの辞書型にはない、いくつかのメソッドを持っています。 加えて、以下の例のように、ドットを使って要素を呼び出すことができます （プログラムをより簡易に書くための機能です）。

>>> foo = Dictm({'a': 1, 'b': 2})
>>> foo.a
1

４パターンの作り方があります。

    1. 辞書を引数に取る方法

>>> foo = Dictm({'a': 1, 'b': 2})

    1. キーワードと引数を取る方法

>>> foo = Dictm(a=1, b=2)

    1. DynaconfのLazySettingsを引数に取る方法

>>> foo = Dictm(Dynaconf(settings_files='settings.toml'))

    1. Dynaconfに対応した設定ファイルのパスを引数に取る方法

>>> foo = Dictm('settings.toml')

If E is present and has a .keys() method, then does: for k in E: D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]
